# Master Thesis — BMC Pipeline from Pitch Decks

Reproducible research artifact for the thesis *Structuring Early-Stage Deal Flow: An Evidence-Grounded BMC Pipeline from Pitch Decks* (Constantin von Krogh, University of Amsterdam).

The project lives in [`Master_Thesis/`](Master_Thesis/). See **[Master_Thesis/README.md](Master_Thesis/README.md)** for setup, pipeline usage, evaluation, and repository layout.

```bash
cd Master_Thesis
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```
